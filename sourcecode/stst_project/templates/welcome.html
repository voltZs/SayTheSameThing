Say the Same Thing

The rules are simple. You say the same thing.

Say the same thing is a turn based game. At first, each player says a random word. If you don't say the same thing, think what connects these two words and say this at your next turn. The game goes on until both of you say the same word.

It is good practice to try and avoid words you've already said!

Say the same thing is a game and app originally developed by the band OK Go. This web app was created by Zsolt Varga as a coursework assignemnt for the Advanced Web Technologies module at Edinburgh Napier University.
